Наш флот

Молодой современный флот состоит из 33 воздушных судов таких типов как: A330, Falcon 7x7. Современный сбалансированный флот, состоящий из наиболее популярных моделей, позволяет сформировать оптимальное предложение для пассажиров вне зависимости от дальности направления перелета. Планы по расширению флота предполагают его дальнейшее увеличение.
